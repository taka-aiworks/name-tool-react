## 🎨 AI漫画制作の流れ

### ステップ1: ネーム制作（このツール）
```
1. ストーリーを入力
2. AIがコマ内容を自動生成
3. 吹き出し・セリフを調整
```

### ステップ2: プロンプト出力
```
4. エクスポートボタンをクリック
5. layout.png + prompts.txt をダウンロード
```

### ステップ3: AI画像生成
```
6. Stable Diffusion / Midjourney を起動
7. prompts.txt からプロンプトをコピペ
8. コマごとに画像を生成
```

### ステップ4: 仕上げ

9. 生成した画像を配置
10. 吹き出し・セリフを重ねる
11. 完成！
